# PySpark Data Engineering Basics

This repository contains basic PySpark examples for learning Apache Spark.

## Technologies
- Python
- PySpark
- Apache Spark
- Jupyter Notebook

## Topics Covered
- Creating SparkSession
- Reading CSV files
- Understanding header and inferSchema
- Basic DataFrame operations

## Dataset
Sample ecommerce dataset with:
- null values
- duplicate records

## Goal
Practice core PySpark concepts used in data engineering pipelines.
