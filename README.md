## 📖 About

This repository is a beginner-friendly reference for working with **Apache Spark using Python (PySpark)**. It covers the most essential data engineering tasks you'll encounter in real-world projects — from loading raw data to building a machine learning pipeline.

Whether you're just starting out with big data or brushing up on Spark fundamentals, this repo has you covered.

---

## 📂 What's Inside

| Topic | Description |
|---|---|
| 📥 **Reading Data** | Load data from CSV, JSON, Parquet, and more |
| 🧹 **Handling Null Values** | Fill, drop, and replace missing data |
| 🔄 **Data Transformations** | Select, filter, rename, and cast columns |
| 📊 **DataFrame Operations** | GroupBy, aggregations, joins, and sorting |
| 🤖 **ML Pipeline** | A basic end-to-end Spark MLlib pipeline |

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Python 3.8+
- Java 8 or 11 (required by Spark)
- Apache Spark 3.x
- PySpark

### Installation

```bash
git clone https://github.com/mohdswalihc/pyspark-data-engineering-basics.git
cd pyspark-data-engineering-basics
pip install pyspark
```
